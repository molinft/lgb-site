# Let's Go Brandon Site

## Linux Setup

Install Node 16 LTS

```sh
sudo apt update
sudo apt upgrade -y

curl -fsSL https://deb.nodesource.com/setup_16.x | sudo -E bash -
sudo apt install -y nodejs
```


## Running

```
npm i
npm run dev
```

Run `npm run list` before commiting. Add ESLint addons for a better experience.
