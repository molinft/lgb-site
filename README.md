# Let's Go Brandon Site

## Setup

Install Node 16 LTS

```sh
sudo apt update
sudo apt upgrade -y

curl -fsSL https://deb.nodesource.com/setup_16.x | sudo -E bash -
sudo apt install -y nodejs
```

## Running

Setup the URL:

```
IMAGE_LOADER_PATH=http://localhost:3000
```

```
npm run dev
```
